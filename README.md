# Система управления контактами на Python

Приложение для управления контактами. 
Программа работает в интерактивном режиме и позволяет пользователю выбирать действия:
- отображение списка контактов;
- добавление новых контактов;
- удаление существующих контактов;

Программа написана с соблюдением всех принципов SOLID
